## Getting Started

### **Q: O que é o CORDEOS?**
A: CORDEOS é um aplicativo de gerenciamento de cifras, que facilita a criação, importação, organização e colaboração em um arranjo musical. Foi desenvolvido com equipes de louvor e musicos individuais em mente.
A: CORDEOS is a digital chord chart management platform that lets you create, organize, and collaborate on musical arrangements. It's designed for individual musicians, worship teams, event organizers, and anyone who works with chord charts and musical performances.

### **Q: How do I download CORDEOS?**
A: CORDEOS is available on:
- **iOS**: Apple App Store
- **Android**: Google Play Store (NOT YET)

### **Q: Is CORDEOS free?**
A: CORDEOS offers a free tier with all core features. Premium features include cloud.

### **Q: Do I need an account to use CORDEOS?**
A: Você pode criar e usar funcionalidades basicas como um convidado, mas para acessar funcionalidades da nuvem é necessário uma conta.
A: You can create and use basic features as a guest, but using cloud features requires an account.

### **Q: How do I create my first account?**
A: 
1. Open CORDEOS
2. Tap "Create New Account" or "Sign Up"
3. Choose: Email/Password or Google Sign-In
4. Fill in your details (email, password)
5. Start creating!

### **Q: What information do I need to provide?**
A: Required for account creation:
- Email address
- Password (if not using Google Sign-In)

### **Q: Can I use CORDEOS offline?**
A: Yes! Once you've downloaded songs to your device, you can access them offline without an internet connection. Cloud features (sharing, sync, etc.) require internet.

## Creating & Managing Songs

### **Q: How do I create my first song?**
A:
1. Go to **Library** (📚 tab)
2. Tap the **+** button (Add Song)
3. Enter song details:
   - Title
   - Author
   - Original Key (musical key)
   - Language
4. Create your first version and sections
5. Tap "Save"

### **Q: What's a "version" of a song?**
A: A version is a different arrangement of the same song. For example:
- "Worship Version" (full arrangement with all sections)
- "Simplified" (fewer chords, easier to play)
- "Acoustic" (modified for acoustic guitar)
- "Training" (teaching arrangement)

You can have unlimited versions per song.

### **Q: How do I add a new version to an existing song?**
A:
1. Open the song
2. Tap "Version" section
3. Tap **+** to add new version
4. Name the version
5. Select "Create from scratch" or "Duplicate existing version"
6. Add sections and save

### **Q: What's a "section"?**
A: A section is a part of your song (Verse, Chorus, Bridge, Intro, etc.) that contains chord charts and lyrics. You can reuse sections multiple times in your song structure.

**Example sections:**
- Verse 1 (V1)
- Verse 2 (V2)
- Chorus (C)
- Bridge (B)
- Intro (I)
- Outro (O)

### **Q: How many sections can I create?**
A: Unlimited! Create as many as needed for your arrangement.

### **Q: Can I reuse sections in one song?**
A: Yes! Your song structure can look like:
`Intro → Verse 1 → Chorus → Verse 2 → Chorus → Bridge → Chorus → Outro`

Notice "Chorus" appears 3 times. You're reusing the same section.

### **Q: How do I delete a song?**
A:
1. Open the song
2. Tap **⋮** (Options menu)
3. Tap "Delete"
4. Confirm deletion

**Warning:** This deletes all versions and sections of the song!

### **Q: How do I duplicate a song?**
A:
1. Open song → **⋮** → "Duplicate"
2. New copy created with same content
3. Edit the copy as needed

### **Q: Can I search for songs?**
A: Yes!
1. Go to Library
2. Use search bar at top
3. Type title, author, or tags

### **Q: How do I add tags to a song?**
A:
1. Open song
2. Tap "Edit"
3. Find "Tags" field
4. Type tag name and press Enter
5. Add as many tags as needed
6. Save song

**Example tags:** #Hymn #HighEnergy #FiveMinutes #Training

### **Q: Can I search by tags?**
A: Yes! Search songs by tags using the search bar in Library.

## Editing & Formatting

### **Q: What is ChordPro format?**
A: ChordPro is a standardized format for writing chords with lyrics. Format: `[Chord]Lyric text`

**Example:**
```
[G]This is a line with [D]a chord above
[Em]And another [Am]line here
```

When displayed, chords appear above lyrics automatically.

### **Q: How do I enter chords in a section?**
A:
1. Open song section
2. Tap "Section Text" field
3. Type lyrics with chords: `[Chord]Lyric`
4. Example: `[Am]Amazing [F]grace, how [C]sweet the [G]sound`
5. Save section

### **Q: What if I don't know how to write ChordPro format?**
A: CORDEOS includes:
- ChordPro format guide (Help → ChordPro Format)
- Video tutorial on entering chords
- Examples in pre-loaded songs
- Auto-formatting suggestions

### **Q: Can I paste text with chords from other apps?**
A: Yes!
1. Copy text from other app (with chords)
2. In CORDEOS, open section
3. Paste into "Section Text" field
4. CORDEOS attempts to parse format
5. Manually adjust if needed

### **Q: How do I edit an existing section?**
A:
1. Open song
2. Tap on section in Song Structure
3. Edit:
   - Section name/type
   - Section code (V1, C, B, etc.)
   - Section color
   - Chord text
   - Notes
4. Tap "Save"

### **Q: How do I delete a section?**
A:
1. Open song
2. In Song Structure, find section
3. Swipe left (mobile) or tap **🗑️** (delete icon)
4. Confirm deletion

### **Q: Can I reorder sections in my song?**
A: Yes!
1. Open song
2. In Song Structure view
3. Tap and hold section → Drag to new position
4. Or use Up/Down arrow buttons
5. Save song

### **Q: What's a section color?**
A: A visual identifier that helps distinguish different section types. When viewing in play mode, each section type shows in its assigned color.


### **Q: What if my lyrics have different punctuation or formatting?**
A: CORDEOS preserves your formatting:
- Line breaks
- Punctuation
- Spacing

Just enter it as you want it displayed.

## Schedules & Services

### **Q: What is a schedule?**
A: A schedule (also called "Service") is a planned musical event with:
- Date and time
- Location
- Team members with assigned roles
- Playlist of songs to perform
- Share code for team members to join

**Perfect for:**
- Worship services
- Concerts
- Rehearsals
- Performances

### **Q: How do I create a schedule?**
A:
1. Go to **Schedule** (📅 tab)
2. Tap **+** (Create Schedule)
3. Enter details:
   - Schedule name
   - Date
   - Start time
   - Location
   - Room/Venue (optional)
   - Annotations/Notes (optional)
4. Select playlist
5. Tap "Create"

### **Q: What are the schedule statuses?**
A: Three statuses track schedule lifecycle:

**Draft** (🔒 Private)
- Not yet shared
- Only you can access
- You can edit freely
- Can be published anytime

**Published** (📢 Active)
- Shared with team members
- Team can access via share code or invite
- Changes sync across devices
- Cannot be unpublished

**Completed** (✓ Past)
- Service date has passed
- Automatically marked when date/time passes
- Changes are no longer synced
- Archived for reference

### **Q: How do I publish a schedule?**
A:
1. Open schedule (in Draft status)
2. Tap **Publish**
3. Confirm: "Make schedule visible to team?"
4. Schedule moves to Published status
5. Team members can now join/view

**Warning:** Once published, cannot be unpublished (but can still edit content).

### **Q: What are roles?**
A: Roles help organize what team member's responsibilities:

- Create role: "Lead Vocalist", "Bassist", "Sound Tech"
- Assign members to roles

### **Q: How do I send team members a share code instead of email?**
A:
1. Open schedule
2. Tap "Share Code" option
3. Copy code (e.g., ABC123XYZ)
4. Share manually via message, email, etc.
5. Team member enters code to join

### **Q: Can I edit a schedule after publishing?**
A: Yes!
1. Open published schedule
2. Tap "Edit"
3. Change songs, times, location, etc.
4. Tap "Save"
5. Changes sync to cloud
6. Team member triggers a reload by going to the schedule tab

### **Q: How do I invite people to a schedule?**
A:
1. In draft schedule, tap "Invite Team"
2. Select roles (or all members)
3. Tap "Send Invites"
4. Email invitations sent
5. Team members receive email with join link

### **Q: What if someone wants to join but wasn't invited?**
A: Two options:

**Option 1**: Share code with them
- Get share code from schedule
- They enter code in "Join Service" section
- They're added to schedule

**Option 2**: Create a new role
- Create new role and add the user email
- invite the role users
- If they already have an account with the email, They're added to schedule

### **Q: Can multiple people edit a schedule at the same time?**
A: No:
- Only the owner can make changes
- Changes sync automatically
- Refresh/reload to see owner's changes

### **Q: How do I delete a schedule?**
A:
1. In Schedule tab, find schedule
2. Tap **⋮** → "Delete"
3. Confirm deletion

**Note:** Can only delete Draft schedules. Published schedules can't be deleted (will be re-synced).

### **Q: Can I duplicate a schedule I was shared?**
A: Yes!
1. In Schedule tab, find schedule
2. Tap **⋮** → "Duplicate"
3. New draft schedule created
4. Edit and customize as needed

## Transposition & Music Features

### **Q: What is transposition?**
A: Changing the musical key of a song without re-entering the chords. All chords automatically shift by the same interval.

**Example:**
- Original key: C
- Transpose up 2 semitones: D
- All chords shift: C→D, G→A, F→G, etc.

### **Q: Why would I transpose a song?**
A: Common reasons:
- Match a singer's vocal range
- Accommodate instrument tuning
- Adapt to different performance situations
- Key compatibility across multiple songs

### **Q: Does transposition affect the saved song?**
A: No! Transposition is temporary during song viewing:
- Original key stays the same
- Transposition applies only to current song
- Reload song = back to original key

### **Q: Can I save a transposed version permanently?**
A: Yes!
1. Transpose song to new key
2. Tap **Save as New Version**
3. Name the version (e.g., "Key: D")
4. New version with transposed key created
5. Original version unchanged

### **Q: Can I set a default key for a song?**
A: Yes!
1. Edit song
2. Find "Music Key" field
3. Set preferred key
4. Save song
5. Song opens in that key by default (doesnt change the content)

## Tips & Best Practices

### **Pro Tips for Musicians:**

**1. Version Strategy**
- Create version for each arrangement
- Name clearly: "Lead", "Simplified", "Training"
- Use "Duplicate" to quickly create similar versions

**2. Section Naming**
- Add descriptive labels: "Verse (Rap)", "Chorus (with harmonies)"

**3. Auto-Scroll Setup**
- Test speed before performance
- "Slow" for learning, "Fast" for energy
- Adjust based on song length and your pace

**4. Backup Strategy**
- Regularly export important songs
- Store backups in cloud/device storage

**5. Team Communication**
- Share code via private message (not public)
- Confirm team members joined before service
- Assign clear roles and responsibilities
